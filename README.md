# Daniell-Element Simulation (Galvanische Zelle)

Eine kleine browserbasierte Webanwendung zur Veranschaulichung des Daniell-Elements im Chemieunterricht (Sekundarstufe II, Grundkurs).

## Inhalt

Die Simulation zeigt fachlich korrekt:

- **Anode (Zink):** Oxidation `Zn → Zn²⁺ + 2 e⁻`
- **Kathode (Kupfer):** Reduktion `Cu²⁺ + 2 e⁻ → Cu`
- **Elektronenfluss im äußeren Stromkreis:** von Zink zu Kupfer
- **Ionenbrücke:** Ladungsausgleich zwischen den Halbzellen

## Lokal starten

1. Repository herunterladen oder klonen.
2. Die Datei `index.html` direkt im Browser öffnen.
   - Es ist kein Build-Prozess notwendig.
   - Es werden keine externen Frameworks verwendet.

## Nutzung mit GitHub Pages

1. Projekt nach GitHub pushen.
2. In den Repository-Einstellungen unter **Pages** als Quelle den Branch (z. B. `main`) und Ordner (`/root`) auswählen.
3. Nach der Veröffentlichung die angezeigte URL öffnen.

Da nur relative Pfade (`style.css`, `script.js`) genutzt werden, ist die Anwendung direkt GitHub-Pages-kompatibel.

## Fachliche Einordnung

Die App ist eine didaktisch vereinfachte Visualisierung eines galvanischen Elements. Sie eignet sich zum Wiederholen der Begriffe **Oxidation**, **Reduktion**, **Elektronenfluss** und **Ionenbrücke**.
